## 🎯 Mi enfoque profesional

```text
Diseñar → Construir → Automatizar → Iterar
```

- **Diseñar:** modelo el dominio antes de escribir la primera línea de código. Schemas normalizados, relaciones explícitas, RLS desde el día uno.
- **Construir:** aplico **Occam's Razor** — la solución más simple que cumple los requisitos suele ser la correcta. Evito sobre-ingeniería.
- **Automatizar:** todo proceso repetitivo es candidato a un workflow. Bash, n8n, scripts: el tiempo es el recurso más caro.
- **Iterar:** los proyectos viven de la retroalimentación. Sprints cortos, métricas claras, mejoras continuas.

## 💼 Proyectos destacados

### 🏋️ FitHub — Aplicación móvil de gimnasio multi-servicio

Marketplace móvil que conecta **gimnasios, entrenadores personales y nutricionistas** en una sola plataforma, permitiendo reservas, planes personalizados y seguimiento de progreso.

> **Stack:** React Native · Expo Router · Supabase (PostgreSQL + Auth + RLS) · Zustand · NativeWind · ClickUp · GitHub  
> **Responsabilidades:** Arquitectura backend completa, modelado de base de datos, configuración de autenticación y políticas RLS, coordinación del equipo y gestión de sprints.

---

### ✈️ Sistema de Gestión de Tiquetes Aéreos — .NET 8 + Arquitectura Hexagonal
Plataforma empresarial para administración integral de aerolíneas con **64 tablas relacionales**, **559 servicios registrados en inyección de dependencias** y mapping de tipos modernos (`DateOnly` / `TimeOnly`).

> **Stack:** .NET 8 · Entity Framework Core · MySQL · Hexagonal Architecture  
> **Logros:** Auditoría completa de namespaces, separación estricta de capas (Domain / Application / Infrastructure), cumplimiento de 4NF en el modelo de datos.

---

### 🤖 Sistema de Justificación de Inasistencias — Automatización con IA
Workflow autónomo end-to-end que recibe justificaciones desde un formulario web, las clasifica con IA y notifica al estudiante por Telegram.

> **Stack:** n8n · OpenRouter (DeepSeek) · Google Sheets · Google Drive · Telegram Bot · Netlify · Docker  
> **Flujo:** Formulario HTML → Webhook → almacenamiento en Drive/Sheets → clasificación con IA (aprobado/rechazado/en revisión) → notificación automática por Telegram con reintentos cron.

---

### 🐙 GitHub MCP Server — Asistente conversacional con IA
Servidor MCP (Model Context Protocol) en Node.js que permite interactuar con la API de GitHub mediante lenguaje natural usando function calling.

> **Stack:** Node.js · Groq (Llama 3.3 70B) · GitHub REST API  
> **Características:** Function calling estructurado, historial conversacional en memoria con trimming por tokens, sanitización de parámetros de entrada.

---

### 🌦️ Sistema Autónomo de Monitoreo Climático
Pipeline automatizado que consulta el clima de Bucaramanga, lo procesa con IA y registra reportes estructurados.

> **Stack:** n8n · OpenWeather API · DeepSeek vía OpenRouter · Google Sheets  
> **Características:** Self-hosted con Docker + ngrok, ejecución programada, decisiones autónomas sobre qué reportar.
